## Purpose

Nate's routing map. Triggers: `operational/commands/nate-commands.md`.

## Where Identity Fits

Nate has no identity — spark holds Brand Studio's. Universal-only principles.

## The Loop

```
funnel-analysis        (AIM — find the leaky bucket; ends in recommended experiments)
      |
      v
experiment-backlog     (QUEUE — ICE-scored, capacity-honest, pre-registered; the discipline)
      |
      v
ab-test-analysis       (JUDGE — the statistics: power, SRM, significance, guardrails →
      |                 ship/extend/stop/investigate)
      v
results log            → wins GRADUATE (rio/pulse/lena/product surface) ·
                         losses archive with cause · lessons compound
```

## Handoff Rules

- **funnel-analysis → backlog**: every recommended experiment enters intake with a falsifiable hypothesis; the biggest leak is the standing first candidate.
- **backlog ↔ ab-test-analysis**: pre-registration enforces the sibling's math *before* launch; analysis delivers the verdict after. One discipline, two moments.
- **muse → nate**: idea intake. **kai ↔ nate**: instrumentation and funnel data in; results into the scorecard; unmeasurable tests are blocked, and the instrumentation need routes to kai.
- **Graduation**: channel wins → rio (standing campaigns); content/hook wins → pulse's register; copy wins → lena; product/site wins → the owning surface (vista's roadmap if it's feature work).
- **board**: experiment spend beyond the envelope gates like any spend. **spark**: experiment *creative* gates like any creative — a test variant is not a gate exemption.
- **vista boundary**: rice-prioritization ranks roadmap features; nate's ICE ranks growth experiments — different altitude; a graduated win that implies feature work crosses to vista's queue.

## Precedence

"Where are we losing users" → funnel. "What should we test next / add this idea" → backlog. "Is this significant / did it win" → ab-test-analysis. Ambiguous → aim, queue, or judge?

## Fallback

No data → funnel runs on operator-supplied numbers or descriptions (estimation labeled); no instrumentation → blocked status, routed to kai; urgency skips the queue, never the discipline.
