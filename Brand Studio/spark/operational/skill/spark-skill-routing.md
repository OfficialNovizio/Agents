## Purpose

Spark's routing map. Triggers: `operational/commands/spark-commands.md`.

## Where Identity Fits

Spark is Brand Studio's leader and carries the active identity (`identity/idea-guardian-david-ogilvy.md`). Identity governs delivery — candid, cited, fix-always-attached; it never changes a threshold, checklist, or verdict. Universal principles win over persona.

## Coach and Gate (the two-skill split)

```
art-direction-critique   (COACH — advisory, upstream, 5-axis scored:
      |                   concept/craft/composition/coherence/finishing →
      |                   SHIP / REWORK / KILL with prioritized concrete fixes)
      v
coherence-qa             (GATE — binary, cited, blocking:
                          atlas kit ✓ · lena voice ✓ · weave arc ✓ · bleed ✓ ·
                          [behavioral — dormant] · coherence judgment →
                          PASS ships / FIX LIST returns)
```

Work should meet the coach before it meets the gate; the gate is not a place to discover concept problems.

## Handoff Rules

- **All outbound creative → coherence-qa.** Pixel's batches, pulse's posts/series, lena's campaigns/sequences, rio's ad creatives — no exemptions, no partial ships.
- **Gate consumes lane results**: atlas audits, lena voice checks + humanic confirmations, weave arc verdicts + ledger refs; missing lanes get run, never waived. Lanes own their lanes; spark owns the whole.
- **Drift and BLEED converge here**: atlas's repeat-violation notes and multi-brand BLEED findings, weave's ARC GAPs, repeat fix-list patterns — spark judges evolution-vs-error and routes amendment questions to the operator. Spark never amends the documents itself.
- **Operator overrides of the gate** are logged like board overrules (precedent's discipline); repeated overrides mean law or gate needs amending.
- **muse → spark** (per catalog): top campaign concepts get a coherence sanity check (coach mode) before development.
- **Cross-department:** board gates decisions, spark gates creative — an ad campaign's spend clears board, its creative clears spark. Behavioral review slots between lena's wording and this gate when that department exists. Echo's investor materials are echo's own lane (marcus/board supervise), but brand-visual elements echo uses still meet atlas's kit.

## Precedence

Submitted-for-ship → gate. "Thoughts on this / make it better" → coach. Both → coach first. Ambiguous → ask which is wanted; the answer changes spark's authority (advice vs block).

## Fallback

No written law in a lane (kit/guide/arc missing) → the gate's power narrows to recommend-only for that lane, stated plainly — and the definer's creation loop is the real fix.
