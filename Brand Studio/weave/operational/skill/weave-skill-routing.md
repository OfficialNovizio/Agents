## Purpose

Weave's routing map. Triggers: `operational/commands/weave-commands.md`.

## Where Identity Fits

Weave has no identity — spark holds Brand Studio's. Universal-only principles.

## Architecture vs Craft (the two-skill split)

```
brand-story-arcs   (ARCHITECTURE — which chapter is this, may it exist:
      |             arc loaded → element advanced → ARC-FIT → continuity → verdict + ledger)
      v
brand-storytelling (CRAFT — how the approved chapter hooks and lands:
                    five-second moment, stakes/elephant, start-in-action,
                    customer-as-hero, dinner-test repeatability, memification)
```

Architecture always runs first: craft polishing an off-arc chapter is wasted craft. Craft runs inside every ON-ARC chapter before it goes to lena for wording.

## Handoff Rules

- **muse → brand-story-arcs**: surviving concepts arrive for chapter positioning; OFF-ARC verdicts return with the on-arc alternative (muse's library records both).
- **brand-story-arcs → brand-storytelling → lena**: approved chapter → craft treatment (hook, moment, stakes) → lena's voice + humanic pass. Weave never wordsmiths final copy.
- **→ pulse**: a chapter serializes into platform-native content; the ledger tracks chapters, pulse's calendar tracks posts. Pulse's series must map to chapters.
- **→ spark**: arc fit is one of spark's three gate references; this skill's verdict is that check's engine. ARC GAP amendments route to the operator via spark's review.
- **↔ echo**: the brand arc and echo's investor pitch-narrative tell the same facts to different audiences — contradictions between them are a finding for both, escalated to the operator.
- **Behavioral Science (dormant)**: chapter framing review (kahneman attention/framing, cialdini ethics line) sits between lena's wording and spark's gate.

## Precedence

"Does this fit our story / what's the story here" → brand-story-arcs. "Make this story better / hook / structure" → brand-storytelling (after arc check if it's campaign-bound). "Build our story" → arc creation loop. Ambiguous → ask whether the need is positioning (architecture) or telling (craft).

## Fallback

No arc → brand-story-arcs' Phase-1 stop (build from real history, or labeled pre-arc content). Craft questions with no campaign attached run brand-storytelling standalone.
