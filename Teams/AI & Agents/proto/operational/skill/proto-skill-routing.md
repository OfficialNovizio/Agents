# proto — Skill Routing (no identity layer — non-leader)

```
idea/pilot (edge, operator, any agent) ─► agent-prototype-kit (manifest)
        └► eval-first-design (criteria, frozen) ─► registry entry (meta, state=prototype)
              └► rapid-prototyping-method (fidelity + timebox, in cage)
                    └► promote-or-archive-verdict (expiry)
                          ├─PROMOTE─► meta (Rail 3 new-agent proposal; full house shape)
                          └─ARCHIVE─► anneal (learnings ledger) + registry state change
```

Handoffs out: meta (registration, standards, governance) · quinn (sandbox policy) · relay (trial tool grants) · anneal (learnings) · gauge (missed-verdict flags) · forge (technique-shaped hypotheses go there instead) · edge (pilots arrive from there).
Precedence: expired verdicts > cage checks > new scaffolds.
