# Cybersecurity Department

**5 agents** — Full-spectrum security: GRC, identity, infrastructure, detection, and data protection. NIST SP 800-30 grounded.

| Agent | Role |
|-------|------|
| **warden** | GRC (CISO persona) — risk register, security policy, vendor AI risk, third-party risk |
| **keyring** | Identity & access management — access reviews, PAM, secrets governance |
| **bastion** | Infrastructure/network security — cloud posture, hardening baselines |
| **cortex** | Detection & response — incident response, threat hunting, security monitoring |
| **veil** | Data protection & privacy — breach notification, DLP, data classification |
