## Purpose

Tempo's routing map. Triggers: `operational/commands/tempo-commands.md`.

## Where Identity Fits

Tempo has no identity — spark holds Brand Studio's. Universal-only principles.

## The Pair

```
usage-licensing   (RIGHTS — verify against the actual use, register, sweep expiries;
      |            ambiguity → operator/counsel, quoted)
      v
sound-identity    (FIT — the sonic guide per brand, derived from the other definers;
                   selections registry-only; drift flagged)
```

Rights gate fit: nothing is selectable until it's cleared. For brands doing no audio content, both are documented no-ops — structure intact, nothing runs.

## Handoff Rules

- **pixel/pulse → tempo**: any produced content carrying audio invokes the pair (selection + license ref travel with the manifest).
- **License spend** → operator, board's gate at its thresholds. **Lapses** flag live uses → operator with options.
- **spark**: audio coherence becomes a gate reference once a brand's sonic guide exists (dormant line until then, coherence-qa's pattern).
- **Jurisdiction/counsel questions** → operator, always.

## Precedence

"Can we use this" → licensing. "What should this sound like" → sound-identity. Both run on any new audio content.

## Fallback

No guide/registry → the build/acquisition loops run first; neutral-labeled or nothing ships in the meantime.
