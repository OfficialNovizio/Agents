## Introduction

coherence-qa is Brand Studio's blocking gate: **every outbound creative passes through it before shipping.** It runs four checks — visual (atlas's brand kit), verbal (lena's voice guide), narrative (weave's arc + ledger), and cross-brand bleed (atlas's separation matrix, multi-brand operators only) — plus a dormant fifth (behavioral review, activates when Behavioral Science is built). The verdict is PASS or an itemized fix list; nothing outbound ships on a fix list. It is the department's version of board's gate, with the same derivative-authority law: spark enforces only what the definers' documents say, and where they're silent, spark recommends rather than blocks.

## Purpose

Individual agents each enforce their own lane well — atlas catches off-palette, lena catches off-voice, weave catches off-arc. What none of them see is the *whole*: an asset that passes each lane separately can still be incoherent as a piece (the visual mood fighting the chapter's tension, a punchy voice register on a somber arc beat), and at production volume someone must be the last pair of eyes with authority to stop the ship. That's spark.

## When to Use

Triggers: "creative review," "final check," "gate this," "ready to ship?" — and **mandatorily** for every outbound creative from any Brand Studio agent (pixel's batches, pulse's posts and series, lena's campaigns and sequences, rio's ad creatives).

Not for: in-progress craft feedback (that's `art-direction-critique`, spark's other skill — REWORK coaching, not gating), or re-running the definers' full audits (spark consumes their results).

## Structure / Protocol

```
Receive the asset + its lane results (atlas audit / lena voice check / weave arc verdict —
run any that are missing rather than gating blind)
  -> Check 1 VISUAL: atlas brand-guidelines PASS in hand?
  -> Check 2 VERBAL: lena voice check clean?
  -> Check 3 NARRATIVE: weave ON-ARC verdict + ledger entry?
  -> Check 4 BLEED (multi-brand only): separation clean?
  -> Check 5 BEHAVIORAL (dormant until dept built): review complete?
  -> COHERENCE JUDGMENT: do the lanes agree with each other as one piece?
    -> PASS (ship + log) / FIX LIST (itemized, cited, blocking) 
```

## Instructions

### Phase 1 — Assemble the Lane Results

An asset arrives with its lane results attached; any missing lane check gets run (by routing to the owning agent's skill), never waived. Spark does not re-litigate a lane's own call — an atlas PASS on palette is final on palette — but spark may send an asset *back* to a lane if new context (the coherence judgment) reveals something the lane couldn't see alone.

### Phase 2 — The Four (Five) Checks

1. **Visual** — atlas's audit verdict, plus spark's spot-check that the audit covered this asset type's applicable kit sections.
2. **Verbal** — lena's voice check block (banned/required patterns, read-aloud), plus the humanic-writing pass confirmation for anything with copy.
3. **Narrative** — weave's ON-ARC verdict with the ledger entry reference; content without narrative positioning ships only as explicitly labeled non-chapter content (routine/functional posts), a category spark tracks so it doesn't quietly become the majority.
4. **Bleed** (2+ brands) — atlas's multi-brand-system verdict; spark is the named recipient of BLEED findings and judges evolution-vs-error, with matrix amendments belonging to the operator.
5. **Behavioral** (dormant) — when Behavioral Science exists: cialdini's principles/ethics review and kahneman's framing check, recorded like any lane. Until then this line reads "not yet in force" in every output — visible, not silent.

### Phase 3 — The Coherence Judgment

The check no lane can run: **as one piece, do the lanes agree?** Named tests: does the visual mood serve the chapter's emotional beat (weave × atlas)? Does the voice register fit the story moment (lena × weave)? Does the layout's focal point land on the message the copy carries (atlas × lena)? Is this piece recognizably from the same brand as last week's output (the portfolio squint test)? Findings here cite *both* documents in tension, and the fix names which lane moves.

### Phase 4 — Verdict, Log, Escalate

- **PASS** — ships. Logged (asset, checks, date) to the gate log; the ledger entry (if a chapter) is confirmed.
- **FIX LIST** — itemized, each item citing its document (kit §, guide rule, arc element, matrix row) or the coherence tension, with the concrete fix and the owning agent. **Blocking**: the asset returns through the fix path and re-gates. No partial ships.
- **Escalations**: document-vs-document conflicts (the kit demands what the arc forbids) go to the operator as amendment questions — spark names the conflict, never resolves the law itself. Repeat fix-list patterns route to the owning definer as drift (their kits/guides may need amending) — same pattern as atlas's drift notes, one level up.

## Principles

- **Nothing outbound skips the gate.** Volume is not an exemption; pixel's batch of forty gates as forty.
- **Derivative authority, like board's.** Spark blocks only on written law (kit/guide/arc/matrix) or named coherence tension between laws — never on taste alone. Taste belongs in art-direction-critique's coaching, upstream.
- **Lanes own their lanes.** Spark doesn't re-audit palettes; it audits the whole.
- **Every fix is owned and actionable.** Item → cited source → fix → owning agent.
- **Conflicts between laws go up.** Spark referees assets, not the documents; the operator amends.
- **Non-chapter content is labeled and counted.** Functional posts are fine; a brand that ships mostly non-chapter content has quietly stopped telling its story — that trend is a finding for weave and the operator.
- **The dormant behavioral line stays visible.** A check that silently doesn't exist never gets built.

## Fallback

- A definer's document doesn't exist yet (no kit/guide/arc) → that lane gates as "no law — recommend only," stated plainly; spark's blocking power extends exactly as far as the written law does.
- Urgent ship request against a fix list → spark doesn't waive; the operator can override, logged like a board overrule (precedent's discipline), and repeated overrides mean the law or the gate needs amending.
- Asset type none of the documents contemplate (new channel, new format) → gate what's covered, name the gap, route the gap to the owning definer.

## Boundaries with Other Skills

- `art-direction-critique` (spark's other skill) is the **coach** — 5-axis scoring, REWORK direction, upstream and advisory. coherence-qa is the **gate** — binary, cited, blocking. Work should meet the coach before it meets the gate.
- Consumes: atlas brand-guidelines + multi-brand-system, lena voice check + humanic pass, weave arc verdicts + ledger. Feeds: the gate log; drift notes to definers; amendment questions to the operator.
- Cross-department: board gates *decisions*, spark gates *creative* — an ad campaign's spend clears board; its creative clears spark; neither substitutes for the other.
