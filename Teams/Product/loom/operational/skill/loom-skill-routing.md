```

```

Handoffs out: spec (test-first verdicts in/out; experiment evidence) · metric (experiment-instrumentation, retention curves) · ux (qualitative why, assumption evidence) · price (revenue experiments; viability) · marcus/vista (PMF verdict → double-down/pivot; roadmap) · proto/forge/scout (shared experiment + adopt-reject patterns) · anneal (overturned verdicts → skill lessons).
Precedence: query the registry before running; criteria frozen before data; a near-miss is a miss; PMF thresholds are flagged rubric.
