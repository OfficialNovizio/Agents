Non-leader: law (the Evidence Gate — ux is the evidence supply) + this routing decide WHICH skill WHEN.

```
research request ─► research-repository (QUERY FIRST) ─verdict─► ANSWERED (cite, no study) | STALE (re-validate) | GAP
                                                                                                   │
                                                              GAP ─► study-design (protocol/method/sample/consent) ─► run
                                                                                                   │ raw data
                                                                          synthesis-discipline (claims + confidence + trace)
                                                                                                   │ filed
                                                                                     research-repository (tagged, cited-by)
support/reviews/NPS ─► voice-of-customer-intake (tag, pattern) ─► repo (directional) | GAP flag ─► study-design
```

Handoffs out: spec (repo IDs = PRD evidence; GAP-driven findings) · loom (assumption checks, qualitative why) · metric (behavioral magnitude is metric's; co-cited) · Client Success/future (read-only intake boundary) · Behavioral Science/future (bias theory dependency, flagged not inlined) · anneal (a shipped miss traced to a stale/over-confident finding → skill lesson).
Precedence: query-first before any study; confidence flag rides every claim; intake is read-only on the support pipeline.
