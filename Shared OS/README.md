# Shared OS-Level Skill Layer

Cross-agent skills that apply to **every agent in every department** — inherited, not copied. Individual agents must not duplicate these into their own skill folders (AGENT-BUILD-PLAYBOOK §7: "Cross-agent capabilities do not go in any individual skill file").

## How inheritance works

- Every agent is bound by every skill in this layer, in addition to its own skills.
- An agent's own operational files may *reference* a shared skill (e.g. an enforcement role) but never re-implement it.
- Precedence: department Security Charter / constitution-grade documents > shared OS skills > agent skills > convenience.
